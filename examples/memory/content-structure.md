Content is structured as follows:

- Module
  - Chapter
    - Page

Every module has at least 1 chapter and page. A module can have multiple chapters. A chapter can have multiple pages. 
For the purpose of this document, each page should be a chunk. A chunk is indicated by # header in markdown. This means that when separating Modules and Chapters, it follow such a structure.

```
# Module

# Chapter 1

# Chapter 1, Page 1

# Chapter 1, Page 2

# Chapter 2

# Chapter 2, page 1

...
```
